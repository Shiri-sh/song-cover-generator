# Song Cover Generator
## Overview
Welcome to the **Song Cover Generator**! This project allows users to quickly and conveniently generate unique cover art for their music compositions and get a full Analysis of the song
With a user-friendly interface and powerful features, creating stunning visuals has never been easier.

## Technologies Used
* **HTML** - for the structure of the web interface.
* **CSS** - for styling the application and ensuring a responsive design.
* **JavaScript** - for implementing dynamic functionalities and enhancing user experience.
* **Node.js** - for handling server-side operations and managing requests.
* **Express** - for creating a robust backend API.
* **GenAI API** - for generate drawing and analysis.
* **GitHub Actions** - for CI/CD to automate deployment and testing processes.
